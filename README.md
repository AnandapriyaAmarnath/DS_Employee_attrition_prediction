# DS_EMPLOYEE_ATTRITION_AND_PREDICTION

## Project Overview
This is a mini project to predict employee attrition using machine learning. It covers:
- Data Preparation
- Exploratory Data Analysis (EDA)
- Data Visualization
- Machine Learning Model (Random Forest)

## Dataset
A synthetic dataset of 500 employees with features like Age, Gender, Department, Education, Job Satisfaction, Monthly Income, Work-Life Balance, and Attrition.

## How to Run
1. Open `DS_EMPLOYEE_ATTRITION.ipynb` in Jupyter Notebook or VS Code.
2. Run all cells sequentially.
3. Visualizations and model evaluation outputs will be displayed inline.
